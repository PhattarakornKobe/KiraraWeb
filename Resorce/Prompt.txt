โปรดระบุสิ่งที่ต้องการให้ปรับปรุงในเว็บไซต์ KiraraWeb:

1) คุณต้องการอะไรจากฉัน? (เลือกหรือพิมพ์อธิบาย)
   - ปรับแต่งโลโก้ / ส่งไฟล์โลโก้ (PNG/SVG)
   - ปรับภาพโปรไฟล์ (pfp) ให้คมชัด/ย่อ/บีบอัด
   - เพิ่มหน้า/เนื้อหา (เกี่ยวกับ, เอกสาร, วิธีใช้)
   - ปรับดีไซน์/แอนิเมชัน
   - แปลภาษา / เพิ่มตัวเลือกภาษา
   - จัดเตรียมไฟล์สำหรับเผยแพร่ (README, GitHub Pages)

2) ฉัน (ผู้ช่วย) ข้อเสนอแนะ/สิ่งที่อยากได้จากคุณ:
   - หากต้องการโลโก้ โปรดอัปโหลดไฟล์ PNG หรือ SVG คุณภาพสูง
   - ถ้ามีโทนสีที่ต้องการ (เช่น #ec4899), ระบุสีหลักและสีรอง
   - ถ้าต้องการลิงก์เชิญ/เซิร์ฟเวอร์/สคริปต์ ให้ส่งลิงก์หรือข้อมูลการตั้งค่า
   - แจ้งข้อความ (bio) ที่ต้องการให้แสดงบนหน้าโปรไฟล์

3) ตัวอย่างคำตอบที่เร็ว:
   - "ปรับโลโก้ — จะอัปโหลด logo.svg"
   - "แปลหน้าเป็นไทย/อังกฤษ"
   - "ต้องการไฟล์ deploy สำหรับ GitHub Pages"

4) หมายเหตุความเป็นส่วนตัว:
   - อย่าส่งข้อมูลลับหรือรหัสผ่านในไฟล์นี้

ตอบในไฟล์นี้หรือส่งไฟล์/รูปภาพโดยตรงในแชท แล้วฉันจะดำเนินการต่อให้ตามคำขอ (เช่น แก้ HTML, ย่อรูป, เพิ่มไฟล์ deploy)
